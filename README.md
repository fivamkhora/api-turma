# API Turma

Backend do projeto Khora/FIVAM desenvolvido com NestJS, PostgreSQL e TypeORM.

## Tecnologias

- NestJS
- Node.js
- TypeScript
- PostgreSQL
- TypeORM

## Instalação

```bash
npm install
```

## Executar em desenvolvimento

```bash
npm run start:dev
```

## Banco de Dados

Configure as variáveis no arquivo `.env`:

```env
DATABASE_HOST=localhost
DATABASE_PORT=5432
DATABASE_USER=khora_user
DATABASE_PASSWORD=sua_senha
DATABASE_NAME=khora
```

## Estrutura Inicial

- Classroom Module
- PostgreSQL Integration
- TypeORM Configuration

- REST API
